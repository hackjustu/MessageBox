# Project Hyla
![](./hyla.jpg)

**Hyla(雨蛙)项目**是关于太阁GitHub小组未来3月的发展规划。

## Motivation
太阁GitHub小组在过去几个月里没有固定的人员，也没有固定的产出模式。为了更好地辅助太阁项目活动的开展，以及更有效地为开源社区布道，我们需要**组建一只高效的行动小组**。

## Goals
- [ ] 明确小组成员组成
- [ ] 明晰小组职责
- [ ] 确定小组的I/O模式
- [ ] 摸索小组的运作模式
